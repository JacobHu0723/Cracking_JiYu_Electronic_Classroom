
# 极域世界(JiYu_World)
破解极域电子教室

<br /><br />
V1.0 特点：运行后立即关闭极域电子教室进程

V2.0 特点：运行175秒后会弹出窗口询问是否关闭。选择“终止”将会在5秒后关闭进程;选择“重试”则重新计时，175秒后再次询问；选择“忽略”退出本极域世界程序。（这个可用于确认老师是否在身边。老师在身边可以点击“重试”或“忽略”，就不会退出。如果老师不在身边，可以点击“终止”退出）

V3.0 特点：运行后等待15秒才会关闭进程（可以用于进去截图）
